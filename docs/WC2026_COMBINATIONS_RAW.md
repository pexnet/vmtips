FIFA Regulations Annex C

## Format

Each of the 495 combinations defines which 3rd-place team goes to which match.
The 8 group winners that play a 3rd-place team are fixed to these matches/sides:
- 1A → match 79, away side
- 1B → match 85, away side
- 1D → match 81, away side
- 1E → match 74, away side
- 1G → match 82, away side
- 1I → match 77, away side
- 1K → match 87, away side
- 1L → match 80, away side
